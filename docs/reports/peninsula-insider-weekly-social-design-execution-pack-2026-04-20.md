## Recommendation

Build the week as **one premium editorial system with five reusable template families**, not seven isolated posts.

That gives the design team three advantages:
- the week feels coherent on-platform
- production stays fast and controlled
- the Friday flagship can carry extra authority without visually breaking the system

The right visual rhythm is:

**cover opener → hero recommendation → utility sequence → comparison chooser → flagship weekend cover → live atmosphere → reflective close**

The week should feel calm, selective, and publishable, not like a batch-made social promo pack.

---

## 1. Weekly production goal

Produce a complete social graphics set for Facebook, Instagram, and LinkedIn that supports the approved weekly narrative arc:

**orientation → taste → utility → decision → action → proof → renewal**

Every asset should help the audience feel one of three things:
- Peninsula Insider knows the Peninsula
- Peninsula Insider has taste
- Peninsula Insider helps me choose well

## 2. Lightweight weekly template system

Use **five base templates** across the full week.

### Template A — Editorial Cover
**Use for:** Monday, Friday, optional Sunday recap cover  
**Job:** set the frame, establish authority, carry the weekly thesis  
**Build:**
- full-bleed or near-full-bleed image
- small kicker or masthead label
- 4 to 6 word headline max
- generous lower-third safe area
- serif headline or strong editorial sans, not both competing

### Template B — Recommendation Hero
**Use for:** Tuesday, optional Facebook Friday crop  
**Job:** let one place or one idea carry the post  
**Build:**
- single-image dominant frame
- optional short recommendation line only
- minimal supporting UI
- stronger image reliance than text reliance

### Template C — Utility Sequence
**Use for:** Wednesday, Friday supporting carousel panels  
**Job:** deliver saveable decision support  
**Build:**
- fixed card grid
- one action line per panel
- consistent label treatment
- optional place tag / route step / timing cue
- photography can alternate with text-led panels

### Template D — Editorial Comparison
**Use for:** Thursday, optional Sunday recap  
**Job:** help readers choose between lanes without feeling like an infographic  
**Build:**
- 2 or 3 evenly weighted columns or panels
- short lane labels only
- restrained divider rules
- quiet tonal differentiation, not loud colour coding

### Template E — Atmosphere Single
**Use for:** Saturday, Sunday  
**Job:** feel observed, current, and close to the ground  
**Build:**
- image-first
- little or no overlay
- one subtle label only if needed
- designed to survive feed, story, and repost crops

### System rule
The whole week should be buildable from these five templates plus crop adaptations. If a daily concept requires an entirely new visual language, it is probably drifting off-brand.

## 3. Core design system for the week

### Typography roles
- **Display / headline:** restrained editorial serif or equivalent high-authority face
- **Utility / labels / metadata:** clean sans serif
- **Rule:** one hero type moment per frame only

### Colour behaviour
Use the established Peninsula Insider palette:
- warm cream / parchment
- stone / sand
- sea-fog blue-grey
- muted olive / eucalyptus
- selective pinot / oxblood accent
- charcoal / ink

Use colour for hierarchy and mood, not urgency.

### Graphic components to standardise once, then reuse
1. **Kicker label**  
   Example use: `This week`, `Weekend edit`, `3 ways to shape the day`
2. **Headline block**  
   Fixed text box style with consistent margins
3. **Place / category tag**  
   Small quiet chip, never loud badge behaviour
4. **Slide number / sequence marker**  
   Only for utility carousels where it aids comprehension
5. **Divider rule**  
   Thin editorial line for comparisons and recap cards
6. **Image frame styles**  
   One single-image style, one split-panel style, one text-card style
7. **Safe margin system**  
   Shared margin logic across 4:5, 1:1, and 1.91:1 variants

### Reuse plan
- Build the headline, kicker, tag, and rule styles as shared components first
- Build Template A to Template E second
- Only then flow in copy and imagery by day

This order will cut rework and keep the week visually tight.

## 5. Recommended file naming system

Use one naming system across master files and exports.

### Format
`pi-social-{weekstart}-{day}-{concept}-{platform}-{ratio}-{asset}-v##.{ext}`

### Example
`pi-social-2026-04-20-mon-week-ahead-ig-4x5-s01-v01.png`

### Field rules
- `weekstart`: `2026-04-20`
- `day`: `mon`, `tue`, `wed`, `thu`, `fri`, `sat`, `sun`
- `concept`: short slug, for example `week-ahead`, `sorrento-base`, `walk-lunch-soak`, `weekend-moods`, `this-weekend`, `peninsula-today`, `week-close`
- `platform`: `ig`, `fb`, `li`
- `ratio`: `4x5`, `1x1`, `191x1`, `9x16`
- `asset`: `s01`, `s02`, `hero`, `summary`, `story01`
- `v##`: `v01`, `v02`, `v03`

### Master working files
Use:
`pi-social-{weekstart}-{day}-{concept}-master-v##.fig`
or equivalent source format.

### Folder structure
- `/social/week-of-2026-04-20/masters/`
- `/social/week-of-2026-04-20/exports/ig/`
- `/social/week-of-2026-04-20/exports/fb/`
- `/social/week-of-2026-04-20/exports/li/`
- `/social/week-of-2026-04-20/source-images/`

## 7. Image sourcing plan by day

### Priority sourcing order
1. Existing Peninsula Insider owned imagery
2. Commissioned or partner-cleared Peninsula photography
3. Verified licensed library images with true location match
4. Controlled generative atmosphere only for non-documentary support, never fake place evidence

### Day-by-day sourcing brief
- **Monday:** 4-image mini library showing week mood, village, coast, table
- **Tuesday:** 2-image Sorrento set with one hero and one detail
- **Wednesday:** 3-image route set with clear progression from movement to meal to unwind
- **Thursday:** 3-image lane set with distinct mood separation
- **Friday:** 4-image premium hero set, strongest curation bar of the week
- **Saturday:** 1 fresh observational still, ideally same-week or same-day feeling
- **Sunday:** 1 reflective still, optional 2-image recap support

### Sourcing quality filter
Reject any image that:
- could plausibly be anywhere else in coastal Australia
- looks too summer-crowded for the current seasonal frame
- feels like tourism-board gloss
- cannot hold text cleanly where overlay is required
- visually cheapens the publication

## 8. Suggested build order for the design team

1. Lock shared components and typography styles
2. Build Template A through Template E master files
3. Source and approve Friday hero imagery first
4. Build Friday flagship cover and carousel shell
5. Build Monday opener using the same cover logic
6. Build Tuesday single-subject recommendation
7. Build Wednesday utility sequence
8. Build Thursday comparison system
9. Build Saturday and Sunday singles last
10. Export all platform variants in one batch and run QA

### Why this order
Friday sets the visual authority for the week. Once that cover language is right, Monday can echo it, Tuesday can narrow from it, and the rest of the week can branch cleanly without drift.

## 9. Designer QA checklist

## Brand fit
- Does this feel editorial, premium, calm, and selective?
- Does it feel unmistakably Peninsula Insider rather than generic destination content?
- Would this sit comfortably beside the site and newsletter without feeling like another brand?

## Editorial usefulness
- Does the asset help the audience orient, choose, save, or act?
- Is the framing actually selective, or is it flattening everything into equal-weight lifestyle fluff?
- Is the message clear in under three seconds?

## Image integrity
- Is the image genuinely Peninsula-relevant and on-angle for the post?
- Is the seasonal feel believable for late April?
- Is it free of brochure gloss and generic stock energy?

## Typography and layout
- Is there one clear focal point?
- Is the text load restrained?
- Are type scale, spacing, and margins consistent with the weekly system?
- Has text-on-image been used only where it improves the asset?

## Platform adaptation
- Has Instagram kept richness?
- Has Facebook been simplified for instant legibility?
- Has LinkedIn been cleaned of consumer-graphic clutter?
- Do all crops protect the essential image subject and headline?

## Production hygiene
- Are file names correct?
- Are source files and exports versioned cleanly?
- Are spelling, place names, and labels correct?
- Are colour profiles and export sizes ready for publishing with no extra intervention?

## 11. Bottom line for the team

The cleanest way to execute this week is to treat it like a short editorial issue, not a social media batch.

Build one visual language. Reuse it intelligently. Give Friday the most authority. Let Wednesday do the utility work. Keep Saturday and Sunday quiet enough to feel lived.

If the result feels like a calm, premium local publication helping people choose the Peninsula well, the pack is right.
